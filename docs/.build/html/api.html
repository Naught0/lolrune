API Reference¶

Clients¶

class lolrune.RuneClient(session: requests.sessions.Session = None)¶

A client which allows you get a champion’s optimal runes.

session : requests.Session, optional
The main session which is used to make all requests. If one is not passed, one will be created.
rune_links : dict

This is the data contained the .data/rune_links.json file. The structure is as follows:

{
    "champion_name": [
        "http://link_to_the_rune.page/",
        "http://a_second_link_if.exists/"
    ]
}
get_runes(champion_name: str) → typing.Tuple[dict]¶

The main method to retrieve optimal runes for a given champion.

champion_name : str
Case insensitive name of the champion to get runes for.
Tuple[dict]

A tuple of dicts which contain the Runeforge data. Below is an example of Runeforge data contained in the dict:

{
    "name": "Varus",
    "title": "Bloodshed Carries a Price",
    "description": "Lethality focused long range poke with [Q].",
    "runes": {
        "primary": {
            "name": "Sorcery",
            "keystone": "Arcane Comet",
            "rest": [
                "Manaflow Band",
                "Celerity",
                "Scorch"
            ]
        },
        "secondary": {
            "name": "Precision",
            "rest": [
                "Triumph",
                "Coup De Grace"
            ]
        }
    }
}
update_champs()¶

A method which updates the .data/rune_links.json file and self.rune_links.

The Runeforge.gg site is frequently updating

class lolrune.AioRuneClient(session: aiohttp.client.ClientSession = None, loop: asyncio.events.AbstractEventLoop = None)¶

An asynchronous version of the RuneClient used to fetch optimal runes for a champ

session : aiohttp.ClientSession, optional
The aiohttp session used in all requests. If none is provided, a new aiohttp.ClientSession will be created.
loop : asyncio.AbstractEventLoop, optional
The asyncio event loop. If none is provided, a new loop will be created.
rune_links : dict

Exceptions¶

exception lolrune.LoLRuneException¶

Base exception for library exceptions.

exception lolrune.RuneConnectionError(status)¶

Raised when a request does not have a status of 200.

status : int
The status of the request which failed.
exception lolrune.ChampNotFoundError(champ)¶

Raised when a champion is not found in the .data/rune_links.json file.

champ : str
The champion which was not found.
